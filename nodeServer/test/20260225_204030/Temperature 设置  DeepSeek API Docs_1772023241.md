# Temperature 设置

`temperature` 参数默认为 1.0。

* 我们建议您根据如下表格，按使用场景设置 `temperature`。

| 场景 | 温度 |
| --- | --- |
| 代码生成/数学解题 | 0.0 |
| 数据抽取/分析 | 1.0 |
| 通用对话 | 1.3 |
| 翻译 | 1.3 |
| 创意类写作/诗歌创作 | 1.5 |
